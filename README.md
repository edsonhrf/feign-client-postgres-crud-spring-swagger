# Feign Client with Postgres CRUD and Swagger Integration

This project demonstrates how to use Feign Client in Java to efficiently consume external APIs, as well as save and manage data in a Postgres database. It implements complete CRUD operations and Swagger integration in Spring Boot 3 to document the API clearly and concisely.

## Key Features

- Usage of Feign Client for consuming external APIs.
- Integration with Postgres database for data storage and management.
- Implementation of complete CRUD operations.
- Detailed API documentation using Swagger.

## Technologies Used

- Java
- Spring Boot 3
- Feign Client
- Postgres
- Swagger

## How to Use

1. Clone this repository to your local environment.
2. Import the project into your favorite IDE.
3. Ensure you have Postgres installed and configured correctly.
4. Run the Spring Boot application.
5. Access the API documentation generated by Swagger for details on endpoints and how to use them.
